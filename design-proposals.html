⭐ A · 极简卡片流

  • 保留紫色渐变头部，品牌色鲜明
  • 横向日期栏，左右滑动切换天数
  • 逾期/今天/已完成 分组展示
  • 底部导航栏：待办 / 统计 / 标签 / 设置
  • 快捷添加 + 下拉选标签一气呵成
  • 适合：喜欢色彩分明、操作直觉的用户

⭐ B · 侧边栏布局

  • 去彩留黑，仅优先级色条保留色彩
  • 筛选 pill 按键（全部/待完成/逾期/按标签）
  • 左侧色条 + 卡片式任务项，一目了然
  • 底部悬浮"新建"大按钮（FAB）
  • 信息密度高，一屏看更多
  • 适合：追求效率、极简风格的用户

⭐ C · 日历集成

  • 周视图方格日历，有任务的日期高亮
  • 逾期任务红色背景突出
  • 左侧标签色条（每标签一种颜色）
  • 底部统计条：完成率 / 已完成 / 待完成 / 逾期
  • 顶行快速输入 + ⋯ 展开更多选项
  • 适合：按日历规划日程的用户

⭐ D · 黑白极简·全览日历 NEW

  • B 的黑白极简视觉 + C 的周历日历网格
  • 左侧优先级色条 + 逾期红色背景卡片
  • 周历日期 + 分组统计一览无余
  • 底部统计条 + FAB 新建按钮
  • 信息密度最高，既看日历又看明细
  • 适合：想要日程总览+任务明细的用户
